Files and assemblies have been modified in Inventor 2019, as such Inventor 2019 or higher is requried to open them. 

The following files are taken from external sites and thus cannot be shared in this git repository. Ensure that the following files are in this folder. Links to the parts are listed along side the file name. 

Download the step file for these components in Autodesk Inventor, convert this to an inventor file - this will happen automatically when opened in Inventor. Save the file as an inventor part file and save to this folder with the part name given on the left hand side of this list.
